Installation & Setup

Follow these steps to enable AI-powered debugging in your workspace.

1

Install VSCode Extension

Search for "MCP Debug Tools" in the VSCode Marketplace. This is required to bridge the AI tool with VSCode's debugger.

VSCode Marketplace
2

Configure MCP Server

Add the following configuration to your AI tool (Cursor or Windsurf settings) to enable the connection.

"mcpServers": {
  "dap-proxy": {
    "command": "npx",
    "args": ["-y", "@uhd_kr/mcp-debug-tools@latest"]
  }
}
üí°

Can't find the extension in your editor?

Forked versions of VS Code (like Cursor or Windsurf) may not show the extension in their specific marketplaces. In this case, you can install it manually using the .vsix file.

A. Download VSIX

Go to the GitHub Releases page and download the latest .vsix file.

View All Releases

B. Install from VSIX

In your editor, go to the Extensions tab, click the (...) menu at the top, and select "Install from VSIX...".
